Gesture Control

用电脑内置摄像头识别手势，零接触控制滚动、鼠标、点击和页面导航。基于 MediaPipe + OpenCV + PyAutoGUI 实现。

Python MediaPipe OpenCV PyAutoGUI Computer Vision
查看 GitHub 仓库

📋 支持的手势

🤚⬆ 向上滚动 手掌向上滑动
🤚⬇ 向下滚动 手掌向下滑动
🖐⬅ 返回上一页 张开手掌向左滑
🖐➡ 前进下一页 张开手掌向右滑
☝️ 移动鼠标 只伸出食指
🤏 点击 拇指和食指捏合

🚀 快速开始

# 克隆仓库
git clone https://github.com/Elevenvenven/Gesture-Control.git
cd Gesture-Control

# 安装依赖
pip install -r requirements.txt

# 运行
python gesture_scroll.py

在这台 Mac 上，--camera 0 是手机连续互通摄像头，--camera 1 对应电脑内置 FaceTime HD Camera。

# 指定摄像头
python gesture_scroll.py --camera 1

⚙️ 调参

# 降低灵敏度
python gesture_scroll.py --threshold 0.12 --cooldown 0.35

# 加快响应
python gesture_scroll.py --threshold 0.06 --frames 5

📦 技术栈

  • MediaPipe Hands — 实时手部关键点检测
  • OpenCV — 摄像头采集和画面叠加
  • PyAutoGUI — 跨平台模拟滚动、鼠标移动和点击
  • NumPy — 手势坐标计算

代码逻辑分为手势识别（食指伸出、五指张开、捏合）和动作执行（滚动/导航/鼠标/点击）两层，通过滑动窗口检测连续帧的方向变化来避免误触，并为每个动作配置独立的冷却时间。

🔧 macOS 权限

第一次运行时，需要授予：

  • 摄像头权限 — 系统设置 → 隐私与安全性 → 摄像头
  • 辅助功能权限 — 系统设置 → 隐私与安全性 → 辅助功能（允许终端/Codex 控制电脑）
