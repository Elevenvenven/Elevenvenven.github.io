1核心(かくしん)语法框架

1.1 句(く)子基本(きほん)構造(こうぞう)：SOV

日(に)语的基本(きほん)语序是 主(しゅ)语(S) + 賓(ひん)语(O) + 謂(い)语(V)，与中(ちゅう)文(ぶん)的SVO不同。

🇨🇳 中(ちゅう)文(ぶん) (SVO)
我 吃 饭。
🇯🇵 日(に)語(ご) (SOV)
私(わたし)は 御(ご)飯(はん)を 食(た)べます。

1.2 判断(はんだん)句(く)：〜は〜です

A は B です（A是B）
私(わたし)は学(がく)生(せい)です。（我是学生。）
これは本(ほん)です。（这是书。）
あそこは駅(えき)です。（那里是车站。）

否定(ひてい)形(けい)：

〜は〜ではありません（不是...）
私(わたし)は学(がく)生(せい)ではありません。（我不是学生。）

過去(かこ)时：

〜は〜でした（曾是...）
昨日(きのう)は月(げつ)曜(よう)日(び)でした。（昨天是星期一。）

1.3 存在(そんざい)句(く)：〜は〜にあります/います

A は B に あります/います（A在B）
あります → 無(む)生(せい)命(めい)物体  |  います → 有(ゆう)生(せい)命(めい)物体（人、動(どう)物(ぶつ)）
本(ほん)は机(つくえ)の上(うえ)にあります。（书在桌子上。）
猫(ねこ)は庭(にわ)にいます。（猫在院子里。）
駅(えき)はどこですか？（车站在哪里？）

1.4 動(どう)詞(し)句(く)：〜は〜を〜ます

主(しゅ)语 は 賓(ひん)语を 動(どう)詞(し)ます
私(わたし)は御(ご)飯(はん)を食(た)べます。（我吃饭。）
彼(かれ)は新(しん)聞(ぶん)を読(よ)みます。（他读报纸。）
私(わたし)は日(に)本(ほん)語(ご)を勉(べん)強(きょう)します。（我学习日语。）

否定(ひてい)：〜ません   過去(かこ)时：〜ました

私(わたし)は肉(にく)を食(た)べません。（我不吃肉。）
昨日(きのう)、映(えい)画(が)を見(み)ました。（昨天看了电影。）

1.5 形(けい)容(よう)詞(し)句(く)

い形(けい)容(よう)詞(し) 直接修饰名词：

大(おお)きい家(いえ) （大的房子）
この林(りん)檎(ご)は甘(あま)いです。（这个苹果很甜。）

な形(けい)容(よう)詞(し) 加「な」修饰名词：

静(しず)かな部(へ)屋(や) （安静的房间）
この公(こう)園(えん)は綺(き)麗(れい)です。（这个公园很美。）

1.6 速成造句法：替换三格

初学阶段不要急着背复杂表达。先把下面三个位置换熟，就能快速说出很多句子。

人 + は + 身份/状态 + です私は会社員です。
把「私」「会社員」换成你自己。
人 + は + 物 + を + 动词ます私はコーヒーを飲みます。
把物品和动作换掉，就能表达日常行为。
地点 + で + 动作カフェで日本語を勉強します。
「で」标记动作发生的地点。
时间 + に + 动作七時に起きます。
具体时间点常用「に」。今天、明天这类相对时间通常不用。
